### Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/ytdlp-gui.git
cd ytdlp-gui
```

2. Install dependencies:
```bash
npm install
```

3. Run in development mode:
```bash
npm run tauri dev
```

4. Build for production:
```bash
npm run tauri build
```
